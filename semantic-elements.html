Semantic Elements

To understand what a Semantic Element in HTML is first we need to define two things; One whatâ€™s a HTML element, and two what is the meaning of the word semantic.
According to W3C an element is the basic building block of HTML and is typically made up of two tags: an opening tag and a closing tag. So basically, all HTML code is made of a bunch of elements.
Some examples of elements are <div><h1><p>, and many more.
Semantic is defined as of or relating to meaning in language by the Merriam Webster Dictionary.
So, when we talk about Semantic Elements in HTML we are referring to an element that has meaning for both the browser and the developer. For example, <article> is a semantic element since it can be understood by both the machine and the person.
